The datalist element is used to define autocompletion values for an associated input element.
Suggested autocompletion values are added to a datalist by nesting one or more option elements between the opening and closing datalist tags.

The bdo element is used override the default directionality of text.
It is used to display characters from languages that are read from right-to-left, such as Hebrew and Arabic.
